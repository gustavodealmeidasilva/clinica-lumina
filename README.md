# Clínica Lumina

Landing page fictícia para uma clínica de estética integrativa, desenvolvida como projeto de portfólio com HTML, CSS e JavaScript puros.

A proposta do projeto é apresentar uma clínica moderna, delicada e confiável, com visual claro, linguagem acolhedora e foco em agendamento de avaliações. A identidade visual usa tons naturais, contraste suave, imagens editoriais e uma composição mais leve para diferenciar o projeto de landing pages mais escuras ou tradicionais.

## Demo

Acesse o projeto publicado:

[https://gustavodealmeidasilva.github.io/clinica-lumina/](https://gustavodealmeidasilva.github.io/clinica-lumina/)

## Objetivo do projeto

Este projeto foi criado para praticar uma landing page comercial completa para o nicho de estética, saúde e bem-estar.

O foco principal foi desenvolver:

- Visual moderno e responsivo
- Estrutura clara para conversão
- Conteúdo comercial organizado
- Menu mobile funcional
- FAQ interativo
- Chamada para agendamento pelo WhatsApp
- Identidade visual diferente de outros projetos do portfólio

## Funcionalidades

- Navegação fixa no topo
- Menu mobile com abertura e fechamento via JavaScript
- Hero editorial com imagem de impacto
- Botões de chamada para avaliação e protocolos
- Bloco de métricas da clínica
- Grade de tratamentos com valores iniciais
- Seção explicando o método de atendimento
- Apresentação da equipe
- Depoimentos de clientes
- FAQ interativo com abertura de respostas
- Formulário visual de contato
- Link de envio pelo WhatsApp
- Botão de voltar ao topo com rolagem suave
- Layout responsivo para desktop, tablet e celular

## Seções da página

### Hero

Apresenta a Clínica Lumina com uma chamada principal, texto de apoio, botões de ação e imagem relacionada ao atendimento estético.

### Métricas

Mostra números de destaque, como avaliações realizadas, protocolos combinados e anos de atuação clínica.

### Tratamentos

Lista protocolos fictícios da clínica, como Glow Facial, Skin Reset, Body Sculpt e Recovery Massage.

### Método

Explica a lógica de atendimento da clínica, passando por diagnóstico, plano e acompanhamento.

### Equipe

Apresenta profissionais fictícias da clínica com imagem, nome e especialidade.

### Resultados

Exibe depoimentos de clientes para reforçar confiança, cuidado e prova social.

### FAQ

Responde dúvidas comuns antes da primeira avaliação, com interação via JavaScript.

### Contato

Mostra endereço fictício, formulário visual e botão para envio de mensagem pelo WhatsApp.

## Como rodar localmente

1. Clone o repositório:

```bash
git clone https://github.com/gustavodealmeidasilva/clinica-lumina.git
```

2. Acesse a pasta do projeto:

```bash
cd clinica-lumina
```

3. Abra o arquivo `index.html` no navegador.

Também é possível usar uma extensão como Live Server no VS Code.

## Como publicar

Este projeto pode ser publicado facilmente em:

- GitHub Pages
- Netlify
- Vercel

## Personalização

Para adaptar o projeto para uma clínica real, altere:

- Nome da clínica
- Endereço
- Número de WhatsApp
- Tratamentos e preços
- Textos institucionais
- Nomes e fotos da equipe
- Depoimentos
- Imagens principais
- Cores e fontes no CSS
