# Clínica Lumina

Landing page fictícia para uma clínica de estética integrativa, criada como peça de portfólio com HTML, CSS e JavaScript puros.

## Destaques

- Layout responsivo para desktop e mobile
- Hero editorial com imagem de impacto
- Seções de tratamentos, método, equipe, depoimentos, FAQ e contato
- Menu mobile com JavaScript vanilla
- FAQ interativo
- Chamada de agendamento via WhatsApp

## Estrutura

```text
clinica-lumina/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── main.js
└── README.md
```

## Tecnologias

- HTML5
- CSS3
- JavaScript vanilla
- Google Fonts
- Imagens remotas do Unsplash

## Uso

Abra o arquivo `index.html` no navegador ou publique a pasta em um serviço como GitHub Pages, Netlify ou Vercel.
